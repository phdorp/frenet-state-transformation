Detailed Description

template<int NumQueries = Eigen::Dynamic>
class FrenetTransform::Transform< NumQueries >

Transformation between Cartesian and Frenet frame. Use the Path properties to implement the transformation independent from the Path implementation.

Template Parameters
NumQueriesnumber of query points with -1 for dynamic point number.
