Detailed Description

template<int NumPoints, int NumQueries = Eigen::Dynamic>
class FrenetTransform::Polychain< NumPoints, NumQueries >

Path representation as polychain. Represents a 2-dimensional path as a polychain. Provide path properties based on finite differences at query points.

Template Parameters
NumPointsnumber of points along the path with -1 for dynamic point number.
NumQueriesnumber of query points with -1 for dynamic point number.
